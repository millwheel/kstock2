# 원페이지 주식투자앱 랜딩페이지 PRD

## 1. 프로젝트 개요

주식투자앱의 핵심 가치를 한 화면 흐름 안에서 명확하게 전달하고, 방문자가 앱 설치 또는 사전예약/회원가입으로 자연스럽게 전환되도록 하는 원페이지 랜딩페이지를 제작한다.

이 랜딩페이지는 단순한 앱 소개 페이지가 아니라, 사용자가 "내 투자 판단을 더 빠르고 선명하게 만들 수 있겠다"는 확신을 얻도록 설계한다.

## 2. 목표

- 주식투자앱의 핵심 기능과 차별점을 1분 안에 이해시킨다.
- 모바일 앱 다운로드 또는 사전예약 전환을 유도한다.
- 초보 투자자와 능동 투자자 모두에게 신뢰감 있는 첫인상을 제공한다.
- 데이터 기반 투자, 관심종목 관리, 실시간 알림, 간편한 포트폴리오 확인이라는 가치를 시각적으로 전달한다.

## 3. 타깃 사용자

### 주요 타깃

- 주식 투자를 시작했지만 정보가 흩어져 있어 판단이 어려운 20~40대 개인 투자자
- 관심종목, 뉴스, 차트, 포트폴리오를 한곳에서 보고 싶은 투자자
- 매수/매도 타이밍을 놓치지 않기 위해 알림 기능을 원하는 사용자

### 보조 타깃

- 기존 증권앱이 복잡하다고 느끼는 초보 투자자
- 매일 아침 또는 장중에 빠르게 시장 상황을 확인하는 직장인 투자자
- 국내주식과 해외주식을 함께 관리하고 싶은 사용자

## 4. 핵심 메시지

### 메인 카피

**흩어진 투자 정보를 한 화면에, 더 빠른 주식 판단을 위한 투자앱**

### 서브 카피

관심종목, 실시간 시세, 주요 뉴스, 포트폴리오 변화를 한눈에 확인하고 중요한 순간을 알림으로 놓치지 마세요.

### 톤앤매너

- 과장된 수익 보장 표현은 사용하지 않는다.
- "수익률 보장", "무조건 상승", "급등주 추천" 같은 고위험 표현을 피한다.
- 신뢰감, 명료함, 속도감, 데이터 기반 판단을 강조한다.

## 5. 페이지 구조

### 5.1 Hero Section

목적: 첫 화면에서 앱의 정체성과 핵심 전환 액션을 즉시 전달한다.

구성 요소:

- 상단 내비게이션: 로고, 기능, 사용 흐름, FAQ, CTA 버튼
- 메인 헤드라인
- 서브 설명문
- 주요 CTA: `앱 다운로드` 또는 `사전예약하기`
- 보조 CTA: `기능 둘러보기`
- 앱 화면 목업 이미지: 관심종목, 포트폴리오, 알림 화면이 보이는 모바일 UI
- 신뢰 지표: 예시로 `관심종목 알림`, `실시간 뉴스 요약`, `포트폴리오 분석`

디자인 방향:

- 첫 화면에서 실제 앱 화면이 명확히 보여야 한다.
- 배경은 과도한 금융 차트 이미지보다 깨끗하고 현대적인 톤을 사용한다.
- CTA 버튼은 한눈에 보이도록 대비를 강하게 준다.

### 5.2 Problem Section

목적: 사용자가 현재 느끼는 투자 정보 탐색의 불편함을 짚는다.

핵심 문구:

- "뉴스는 따로, 차트는 따로, 내 종목은 또 따로 보고 있나요?"
- "중요한 공시와 가격 변동을 뒤늦게 확인하고 있나요?"

구성 요소:

- 3개의 문제 카드
  - 정보가 여러 앱에 흩어짐
  - 종목 변동을 실시간으로 확인하기 어려움
  - 포트폴리오 상태를 직관적으로 보기 어려움

### 5.3 Solution Section

목적: 앱이 문제를 어떻게 해결하는지 간결하게 보여준다.

핵심 기능:

- 관심종목 통합 대시보드
- 실시간 가격 변동 및 목표가 알림
- 종목별 뉴스/공시 요약
- 포트폴리오 수익률 및 비중 시각화

표현 방식:

- 기능별 앱 화면 캡처 또는 목업을 함께 배치한다.
- 기능 설명은 짧고 구체적으로 작성한다.
- 각 기능은 "사용자가 얻는 이점" 중심으로 설명한다.

### 5.4 Feature Highlights

목적: 앱의 주요 기능을 신뢰감 있게 정리한다.

기능 카드:

1. **내 종목 한눈에 보기**
   - 관심종목의 현재가, 등락률, 거래량, 주요 뉴스를 한 화면에서 확인한다.

2. **중요한 순간만 알림**
   - 목표가 도달, 급등락, 뉴스 발생 등 사용자가 설정한 조건에 따라 알림을 제공한다.

3. **포트폴리오 자동 분석**
   - 보유 종목 비중, 평가손익, 섹터 편중을 시각적으로 보여준다.

4. **시장 흐름 요약**
   - 장 시작 전과 장 마감 후 주요 지수, 인기 종목, 핵심 뉴스를 요약한다.

### 5.5 How It Works

목적: 앱 사용 흐름을 단순하게 설명해 진입 장벽을 낮춘다.

단계:

1. 관심종목을 추가한다.
2. 목표가와 알림 조건을 설정한다.
3. 뉴스, 시세, 포트폴리오 변화를 한 화면에서 확인한다.
4. 중요한 변화가 생기면 알림을 받고 판단한다.

### 5.6 Trust Section

목적: 금융 서비스에서 중요한 신뢰와 안전성을 전달한다.

포함 내용:

- 투자 판단 보조 서비스이며 투자 결과를 보장하지 않는다는 안내
- 개인정보 및 계좌 연동 정보 보호 정책 요약
- 알림과 데이터 출처에 대한 투명한 설명
- 사용자 리뷰 또는 베타테스터 피드백 영역

주의:

- 수익 인증, 과장된 후기, 특정 종목 추천처럼 오해될 수 있는 요소는 지양한다.

### 5.7 Final CTA Section

목적: 페이지 하단에서 다시 한번 전환을 유도한다.

구성 요소:

- 짧은 전환 카피: "내 종목의 중요한 순간을 놓치지 마세요"
- CTA 버튼: `지금 시작하기`, `앱 다운로드`, `사전예약하기`
- 앱스토어/구글플레이 배지 또는 사전예약 폼

### 5.8 FAQ Section

질문 예시:

- 이 앱은 주식 매매 기능을 제공하나요?
- 국내주식과 해외주식을 모두 지원하나요?
- 알림 조건은 직접 설정할 수 있나요?
- 포트폴리오 연동 시 개인정보는 안전한가요?
- 무료로 사용할 수 있나요?

## 6. 디자인 가이드

### 전체 인상

- 신뢰감 있는 핀테크 서비스
- 복잡한 주식 정보를 쉽게 정리해주는 분석 도구
- 초보자도 접근 가능한 명료한 인터페이스

### 컬러 방향

- 기본 배경: 화이트 또는 아주 밝은 그레이
- 메인 컬러: 신뢰감을 주는 딥 그린, 네이비, 블루 계열 중 하나
- 상승/하락 표시: 국내 투자자에게 익숙한 빨강/파랑 체계를 사용하되 과도하게 자극적이지 않게 조정
- CTA 컬러: 메인 컬러와 명확히 구분되는 고대비 색상

### 타이포그래피

- 헤드라인은 굵고 짧게 구성한다.
- 금융 데이터 영역은 숫자 가독성이 좋은 폰트를 사용한다.
- 본문은 모바일에서도 읽기 쉬운 크기와 줄간격을 유지한다.

### 이미지/비주얼

- 실제 앱 화면처럼 보이는 모바일 목업을 핵심 비주얼로 사용한다.
- 차트, 포트폴리오 원형 그래프, 알림 카드, 뉴스 요약 카드 등 기능을 직접 보여주는 UI를 활용한다.
- 추상적인 그래픽보다 사용자가 실제로 보게 될 화면을 우선한다.

### 레이아웃

- 모바일 퍼스트로 설계한다.
- 데스크톱에서는 최대 콘텐츠 폭을 제한해 읽기 쉽게 구성한다.
- CTA는 첫 화면, 중간 기능 설명 이후, 마지막 섹션에 반복 배치한다.
- 카드형 정보는 2~4개 단위로 정리해 빠르게 스캔할 수 있게 한다.

## 7. 콘텐츠 원칙

- 사용자의 투자 불안감을 자극하기보다 정보 정리와 판단 보조의 가치를 강조한다.
- 기능 설명은 "무엇을 제공하는가"보다 "사용자가 무엇을 더 쉽게 할 수 있는가"에 초점을 둔다.
- 투자 수익을 보장하거나 특정 종목 매수를 유도하는 문구는 사용하지 않는다.
- 모든 금융 관련 문구에는 신뢰성과 책임감을 유지한다.

## 8. 전환 목표 및 KPI

주요 KPI:

- CTA 클릭률
- 앱스토어/구글플레이 이동률
- 사전예약 폼 제출률
- 페이지 스크롤 75% 이상 도달률
- FAQ 섹션 도달률

보조 KPI:

- Hero Section 이탈률
- 기능 섹션 체류 시간
- 모바일 전환율

## 9. 필수 구현 요구사항

- 반응형 웹 지원
- 모바일 360px 너비에서도 CTA와 앱 목업이 겹치지 않아야 함
- CTA 버튼은 모든 주요 뷰포트에서 명확히 보여야 함
- 접근성 고려: 버튼 대비, 폼 라벨, 키보드 탐색, 이미지 대체 텍스트
- 페이지 로딩 속도 최적화
- 앱스토어/구글플레이 링크 또는 사전예약 폼 연결
- 투자 유의 문구 포함

## 10. 투자 유의 문구

랜딩페이지 하단 또는 CTA 인근에 아래와 같은 안내 문구를 포함한다.

> 본 서비스는 투자 정보를 제공하고 투자 판단을 보조하기 위한 서비스입니다. 특정 금융상품의 매수 또는 매도를 권유하지 않으며, 투자 결과에 대한 책임은 투자자 본인에게 있습니다.

## 11. 성공 기준

- 첫 화면만 보고도 앱의 핵심 가치가 이해된다.
- 사용자가 기능을 읽지 않아도 앱 화면 목업을 통해 사용 목적을 파악할 수 있다.
- CTA가 자연스럽고 반복적으로 노출된다.
- 금융 서비스로서 신뢰감과 책임감이 느껴진다.
- 원페이지 구조 안에서 문제 제기, 해결책, 기능, 신뢰, 전환 흐름이 끊기지 않는다.
